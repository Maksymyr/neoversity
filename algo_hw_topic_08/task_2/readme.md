# Завдання 2 — сума всіх значень у BST

Функція, що рахує суму всіх значень у двійковому дереві пошуку (BST).

## Запуск

```bash
python find_sum.py        # демо на дереві з 8 вузлів
python test_find_sum.py   # тести
```

## Реалізація

- `Node` / `BST` — та сама мінімальна реалізація BST, що й у task_1.
- `find_sum(tree)` — рекурсивний обхід (DFS): для кожного вузла повертає `value + find_sum(left) + find_sum(right)`. Порядок обходу (in-order / pre-order / post-order) для суми не має значення.
- Складність — `O(n)` за часом і `O(h)` за пам'яттю (стек рекурсії).
- Пуста tree → `0`.
