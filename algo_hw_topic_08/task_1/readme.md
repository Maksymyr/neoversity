# Завдання 1 — найменше значення в BST

Функція, що знаходить найменше значення у двійковому дереві пошуку (BST).

## Запуск

```bash
python find_min.py        # демо на дереві з 8 вузлів
python test_find_min.py   # тести
```

## Реалізація

- `Node` / `BST` — мінімальна реалізація BST з `insert` (без обробки дублікатів — у дереві унікальні значення).
- `find_min(tree)` — використовує властивість BST: **найменше значення завжди в лівому крайньому вузлі**. Спускаємось ліворуч від кореня, доки не дійдемо до вузла без лівого нащадка. Складність — `O(h)`, де `h` — висота дерева (`O(log n)` для збалансованого, `O(n)` для виродженого).
- Пуста tree → `ValueError`.
