# Завдання 2 — сніжинка Коха

Рекурсивна побудова фракталу «сніжинка Коха» через `turtle`. Рівень рекурсії задає користувач.

## Запуск

```bash
python koch_snowflake.py <level> [--size 300]
# без аргументів -> програма запитає рівень рекурсії інтерактивно
```

## Тести

```bash
python test_koch_snowflake.py
```

Тести мокають `turtle.Turtle` і перевіряють інваріанти кривої Коха:
- кількість прямих сегментів = `4**level`;
- сумарна довжина = `L · (4/3)^level`.

## Реалізація

- `koch_segment(t, length, level)` -> базовий випадок (`level == 0`) малює прямий відрізок; інакше ділить його на 3 частини й рекурсивно малює чотири ламані `_/\_` з поворотами 60°/-120°/60°.
- `draw_snowflake(level, size)` -> тричі викликає `koch_segment` з поворотом на 120° — три сторони сніжинки.
- Рівень читається з CLI (`argparse`) або через `input()`, якщо аргумент не передано.
- Рекомендований діапазон рівнів — `0..7`. При `level > 7` програма попереджає і запитує підтвердження `[y/N]`. При `level == 0` малюється просто рівносторонній трикутник (база рекурсії).
