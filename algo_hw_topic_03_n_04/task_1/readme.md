# Завдання 1 — рекурсивне сортування файлів

Скрипт рекурсивно обходить вихідну директорію, копіює файли у директорію призначення та групує їх у піддиректорії за розширенням.

## Запуск

```bash
python sort_files.py <source> [destination]
# дефолт — ./dist
```

## Тести

```bash
python test_sort_files.py
```

## Реалізація

- `parse_args` -> `argparse` з обов'язковим `source` і опціональним `destination` (default = `Path("dist")`).
- `process_directory(source, destination)` -> рекурсивно обходить дерево: для піддиректорій викликає себе, для файлів викликає `copy_file`.
- `copy_file(file_path, destination)` -> формує піддиректорію за розширенням (нижній регістр, без крапки; файли без розширення `no_extension`), копіює `shutil.copy2` зі збереженням метаданих.
- Захист від циклів: пропускаються симлінки та сама тека призначення (порівняння через `Path.resolve()`).
- Обробка винятків: `PermissionError`, `OSError`, `shutil.SameFileError` -> логуються червоним і не зупиняють обхід.
